## Tags
*Branches should all start with one of the below tags*

| Tag       | Description                    | Example |
| --------- | ------------------------------ | ------- |
| feat/     | implementing a new feature     |         |
| refactor/ | refactoring existing features  |         |
| fix/      | small bug patches              |         |
| ARCHIVE/  | an old version we wanted saved |         |
